# TikTok Food Direction

Use when building or revising logical scenes and Seedance segments.

## Core Rules

- The first 1-3 seconds must stop the scroll.
- Opening must be texture-first and paid off immediately.
- Use 75-80% detail shots, 20-25% context shots.
- A context shot must prepare the next detail payoff.
- Every 3-5 logical scenes needs a texture payoff or strong material contrast.
- Avoid two context shots in a row unless necessary.
- Keep the **2D illustrated** Licorn character as a support for action and final satisfaction, not the main subject in every shot. Never prompt a 3D/CGI mascot reinterpretation.
- Compress utility steps aggressively; linger only on payoffs.

## Strong Food Gestures

Prefer:

- pouring;
- cracking;
- piping;
- glazing;
- spreading;
- dusting;
- sizzling;
- steaming;
- revealing;
- filling;
- lifting ribbons;
- cutting only when the axis and state are simple.

## Anti-Patterns

- neutral mise en place;
- final-dish slicing as hook;
- too many medium shots;
- action that requires exact causal mechanics;
- multiple hands doing different things;
- tiny garnish drift;
- oven door action plus another action in the same shot.
