# Failure States

Use this list to add negatives before generation when ambiguity is high.

## Scale and proportion failures

- food item appears unrealistically huge relative to hands, plate, or island;
- food item appears unrealistically tiny for the intended dish;
- object size changes between contiguous segments without narrative reason;
- repeated elements (balls, domes, rosettes) drift in size within one shot.

## Material failures

- caramel bends like syrup when it should crack;
- cream becomes watery or grainy when it should hold shape;
- glaze turns opaque paste when it should be smooth and reflective;
- choux topology collapses into one smooth mass.

## Causality failures

- tools or ingredients move without visible agent hands;
- movement starts from a final position instead of a plausible start position;
- induction shows visible flame or glow.

## Prompting tactic

For each high-risk scene, include:

1. one positive material target;
2. one scale lock;
3. one explicit "must not" failure.
